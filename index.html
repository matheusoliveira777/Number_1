Sobre o RGB e CMYK

RGB" é a abreviatura do sistema de cores aditivas formado pelos tons de Vermelho (Red), Verde (Green) e Azul (Blue). O propósito principal do sistema RGB é a reprodução de cores em dispositivos eletrônicos como monitores de TV, computadores, mídias digitais, celulares em geral

RGB no Pixel

Cada pixel pode ser controlado para exibir uma cor específica, e essa cor é determinada pela combinação das intensidades das cores primárias RGB (Vermelho, Verde, Azul).Por outro lado, RGB refere-se ao sistema de cores aditivo que permite a criação de uma vasta gama de cores através da mistura das cores primárias.

O que é CMYK

CMYK é uma abreviação do sistema formado pelas cores Ciano, Magenta, Amarelo (Yellow) e Preto (Black). Ele é usado em camisas, convites, faixas, entre outros materiais impressos.

RGB x CMYK

Os dois sistemas de cores funcionam de forma totalmente oposta. Enquanto o RGB é formado pela adição de luz, o CMYK funciona pela subtração da luz. Isso acontece porque as cores RGB são criadas pela emissão de pontos luminosos a partir de uma TV por exemplo, enquanto que as cores CMYK são formadas por pigmentos ( tintas ) adicionados no papel.
